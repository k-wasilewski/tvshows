# Travel Agency in a Box - end to end tests

## How to run

### interactive mode

To run use npm

```CMD
npx cypress open
```

or yarn command

```CMD
yarn run cypress open
```

or use custom script

```CMD
npm run cypress:open
```

### Command Line

Run with a browser

```CMD
yarn run cypress run --browser chrome
```

Run selected spec

```CMD
yarn run cypress run --spec "cypress/integration/shared/**/*"
```
